# Example Skill

This is a minimal skill that gets loaded into the system prompt.
Skills are Markdown files in the workspace/skills/ directory.

## What you can do here:
- Document workflows
- Define coding conventions
- Set project-specific rules
- Provide step-by-step procedures

When a user's request matches a trigger keyword, the skill content
is included in the system prompt automatically.
